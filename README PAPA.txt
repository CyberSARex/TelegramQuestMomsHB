1. ПКМ по папке "DRMamo4ki"
2. Open with Code (VSCode)
3. Выбери файл "main.py"
4. Ctrl + Shift + P
5. Python: Select Interpreter
6. Если есть кнопка "Python 3.11.2 ('HBTeleVenv': venv)", то нажми её
   Если нет кнопки, то "Enter interpreter path..." и выбрать "HBTeleVenv\Scripts\python.exe
7. Ctrl + F5 по main.py (Запустить main.py)

И всё работает, можно писать '/start' в бота @cybersarexquest_bot
